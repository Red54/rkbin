# RK3528 Release Note

## rk3528_ddr_1056MHz_wdqs_{2L_PCB}_{LP4_4x_eyescan}_v1.12.bin

| Date       | file                                                        | Build commit | Severity  |
| ---------- | ----------------------------------------------------------- | ------------ | --------- |
| 2025-11-20 | rk3528_ddr_1056MHz_wdqs_{2L_PCB}_{LP4_4x_eyescan}_v1.12.bin | 7919c80db0   | important |

### Warn

1. When there is no DQSn pull-up resistor on the PCB, only the regular DDRBIN in the bin/rk35 directory can be used. When there is a DQSn pull-up resistor on the PCB, only this version of DDRBIN can be used. Otherwise, there may be reliability issues.

### New

1. Add WDQS support and it must be used in conjunction with `bin/rk35/rk3528_bl31_v1.21.elf` or later versions. At the same time, a 560 ohm weak pull-up resistor is required on DQSn on the PCB.
