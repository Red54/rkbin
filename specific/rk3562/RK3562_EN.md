## rk3562_ddr\_{1560...920}MHz_wdqs\_{LP4_4x_eyescan}_v1.08.bin

| Date       | file                                                         | Build commit | Severity  |
| ---------- | ------------------------------------------------------------ | ------------ | --------- |
| 2025-11-20 | rk3562_ddr\_{1560...920}MHz_wdqs\_{LP4_4x_eyescan}_v1.08.bin | 97267538c8   | important |

### Warn

1. When there is no DQSn pull-up resistor on the PCB, only the regular DDRBIN in the bin/rk35 directory can be used. When there is a DQSn pull-up resistor on the PCB, only this version of DDRBIN can be used. Otherwise, there may be reliability issues.

### New

1. Add WDQS support. And a 560 ohm weak pull-up resistor is required on DQSn on the PCB.
